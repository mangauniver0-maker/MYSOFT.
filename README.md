# 🚀 MYSOFT — Site Officiel

Bienvenue sur le dépôt du site web officiel de **MYSOFT**, une startup innovante spécialisée dans le développement de solutions numériques.

---

## 🌐 À propos de MYSOFT

**MYSOFT** est une startup qui vise à créer des solutions digitales simples, efficaces et accessibles.
Ce site représente notre vitrine en ligne pour présenter notre équipe, notre histoire et nos services.

---

## 🛠️ Technologies utilisées

Ce site a été développé uniquement avec :

* HTML5
* CSS3

👉 Aucun framework ni JavaScript utilisé pour le moment.

---

## 📁 Structure du projet

```bash
mysite/
│── index.html
│── equipe.html
│── historique.html
│
│── main.css
│── equipe.css
│── historique.css
```

---

## 📄 Pages du site

* **index.html** → Page d’accueil
* **equipe.html** → Présentation de l’équipe
* **historique.html** → Histoire de MYSOFT

---

## 📩 Formulaire

Le formulaire utilise **FormSubmit (Formt)** pour envoyer les messages sans backend.

👉 Cela permet :

* d’envoyer des emails directement
* sans serveur ni code backend

1. Télécharge ou clone le projet :

```bash
git clone https://github.com/ton-username/mysite.git
```
